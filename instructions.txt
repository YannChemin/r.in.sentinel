r.in.sentinel is a GRASS GIS Addons module, see $HOME/dev/grass for grass gis source tree, $HOME/dev/grass-addons for GRASSGIS Addons source tree. This module uses cubo library to connect to Microsoft Planetary Computer (direct access), or GEE (needs credentials), take input parameters from the user, use g.region (or read directly the GRASS GIS Mapset computational region) to set the download boundaries. Downloaded datasets are grouped (see i.group GRASS GIS module) by date once downloaded and imported into the working Mapset.
Create full complement of GRASS GIS Addons files required to make it function properly. That also includes a testsuite where a small temporary download of Plumergat (56400, France) will be done to evaluate the actual download works and is grouped and inserted correctly into the mapset.
For Sentinel-2 import, propose a cloud removal algorithm application directly using specific bands and some algorihm.
